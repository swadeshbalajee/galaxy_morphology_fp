
# User Manual

1. Open the frontend at `http://localhost:8501`.
2. Upload a galaxy image.
3. Click **Predict Morphology**.
4. Read the predicted class and confidence bars.
5. If the prediction is wrong, enter the correct label in the feedback section.
6. Open the Pipeline Console page to inspect Airflow, MLflow, Prometheus, Grafana, and Loki.
7. Review the latest generated report shown in the UI.
